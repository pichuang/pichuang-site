Jekyll offers powerful support for [code snippets][code-snippet] -- [Rouge][rouge]:

{% highlight ruby %}
def print_hi(name)
  puts "Hi, #{name}"
end
print_hi('Tom')
#=> prints 'Hi, Tom' to STDOUT.
{% endhighlight %}
